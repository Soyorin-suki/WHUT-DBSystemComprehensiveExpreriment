# 教学管理信息系统--学生选课及课程安排数据库综合实验
Teaching Management Information System--Comprehensive Experiment on Student Course Selection and Course Arrangement Database
下简称TMIS.

该项目为武汉理工大学数据库综合实验，选题为TMIS。

该项目实现了一个教学管理信息系统（教学管理 + 学生选课 + 课程安排），包含：用户管理（学生/教师/管理员）、课程与课表管理、选课（含冲突检查与容量控制）、成绩录入与查询、后台管理界面，以及前端页面和若干典型 SQL/事务/触发器/视图。
